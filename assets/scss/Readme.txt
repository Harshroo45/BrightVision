SCSS Stylesheets
================

This folder contains the SCSS source files for the website styles.

These files are compiled into the CSS files in the ../css/ folder.

STRUCTURE:
- Various SCSS files organized for maintainability
- Compiled to main.css for production use

To modify styles, edit the SCSS files and recompile using a SASS compiler.
