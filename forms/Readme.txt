Contact Form Handler
====================

This folder contains the PHP scripts for handling contact form submissions.

FILES:
- contact.php - Handles contact form submissions
- get-a-quote.php - Handles quote request form submissions

REQUIREMENTS:
- PHP enabled on the server
- Email functionality configured on the server

CONFIGURATION:
Replace 'contact@example.com' in the PHP files with your actual email address.
